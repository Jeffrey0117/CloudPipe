# LurlHub 頁面規格書

## 設計原則
- **風格**：輕鬆幽默、不要太正經
- **規模**：精簡實用、不過度設計
- **調性**：像朋友推薦好工具，不像企業賣產品

---

## 頁面結構

```
/lurl/
├── /                    # Landing 首頁
├── /pricing             # 價格方案
├── /guide               # 使用教學
├── /download            # 腳本下載
├── /feedback            # 意見回饋
├── /browse              # 瀏覽內容（現有）
├── /view/:id            # 單一內容（現有）
└── /member/*            # 會員區（見會員規格）
```

## 實作優先序

1. **Landing `/`** - 第一印象，必須先有
2. **Download `/download`** - 轉換入口
3. **Pricing `/pricing`** - 商業核心
4. **Guide `/guide`** - 降低流失
5. **Feedback `/feedback`** - 收集回饋

## 技術備註

- 所有頁面沿用現有 lurl.js 內嵌 HTML 方式
- 風格延續現有深色主題
- RWD 響應式（現有 breakpoint）
- 共用 Header/Footer 元件化
