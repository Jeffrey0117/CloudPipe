## Phase 1：管理員標註

**標註位置：**
1. `/lurl/browse` - 瀏覽頁卡片
2. `/lurl/view/:id` - 詳情頁

**UI 設計：**
```
┌─────────────────────────────────┐
│  [影片/圖片縮圖]                  │
│                                  │
│  標題：xxxxx                      │
│                                  │
│  [奶子] [屁股] [鮑魚] [全身]       │  ← 點擊切換
│  [姿勢] [口交]                    │
└─────────────────────────────────┘
```

**操作：**
- 點擊標籤 = 切換選中/取消
- 即時保存（不用按確認）
- 需要管理員登入

## 技術實作

### 資料結構

記錄加 `tags` 欄位：
```json
{
  "id": "xxx",
  "title": "...",
  "tags": ["奶子", "屁股"],
  ...
}
```

### API

| 端點 | 說明 |
|------|------|
| `PATCH /lurl/api/records/:id/tags` | 更新標籤（需 Admin） |
| `GET /lurl/api/tags` | 取得所有可用標籤 |

### 瀏覽頁篩選

- 加標籤篩選器
- 可多選
