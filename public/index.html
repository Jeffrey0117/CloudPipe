Deploy Modes

API Services (Path-based)

上傳 .js 檔案，透過路徑存取 API

URL: epi.isnowfriend.com/{name}

Examples
  • YouTube 下載 API → /ytdownload
  • 圖片壓縮 API → /compress
  • 翻譯服務 → /translate
  • Webhook 接收器 → /webhook

App Deploy (Subdomain)

上傳 .zip 專案，自動建立子域名

URL: {name}.isnowfriend.com

Examples
  • 個人 Blog → blog.isnowfriend.com
  • 作品集網站 → portfolio.isnowfriend.com
  • React/Vue App → myapp.isnowfriend.com
  • 文件站 → docs.isnowfriend.com

API Service Format

上傳的 .js 檔案需 export 以下格式：

module.exports = {
  match(req) {
    return req.url.startsWith('/yourname');
  },
  handle(req, res) {
    res.writeHead(200, { 'content-type': 'application/json' });
    res.end(JSON.stringify({ hello: 'world' }));
  }
};
